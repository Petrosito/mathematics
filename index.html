Арифметика

Арифметика - это раздел математики, изучающий основные операции с числами: сложение, вычитание, умножение и деление...

Основные операции

  • Сложение: a + b = c
    Например: 5 + 3 = 8
  • Вычитание: a - b = c
    Например: 10 - 4 = 6
  • Умножение: a ⋅ b = c
    Например: 2 ⋅ 7 = 14
  • Деление: a / b = c
    Например: 15 / 3 = 5

Свойства операций

  • Переместительное свойство: a + b = b + a, a ⋅ b = b ⋅ a
  • Сочетательное свойство: (a + b) + c = a + (b + c)

Примеры

Задача: Найти сумму чисел 12 и 7. Ответ: 19.
